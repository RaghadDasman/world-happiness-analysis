# 🌍 World Happiness Analysis 😊

This project explores the **World Happiness dataset (2015–2019)** to understand what factors influence happiness across countries.

---

## 🎯 Project Goal

In this project I analyze global happiness data to:

- 📊 Explore happiness trends across countries
- 💰 Study the relationship between happiness and GDP
- 🤝 Understand the impact of social support
- 🕊️ Examine the role of freedom and life expectancy
- 🤖 Build a machine learning model to predict happiness scores

---

## 💡 Key Insights

- 🌍 Countries with higher **GDP** tend to report higher happiness scores.
- 🤝 **Social support** shows a strong relationship with happiness.
- ❤️ **Life expectancy** also correlates with higher well-being.
- 📈 Happiness levels remain relatively **stable across years**.

---

## 📊 Dataset

World Happiness Report dataset (2015–2019).

It includes factors such as:

- Happiness score  
- GDP per capita  
- Social support  
- Life expectancy  
- Freedom  
- Generosity  
- Perception of corruption
